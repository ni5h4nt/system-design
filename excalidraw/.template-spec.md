# Excalidraw Interview Diagram Template Specification

## Structure (top to bottom)

Each diagram follows this vertical layout at x=50-1250, full width 1200px:

### 1. Title + Subtitle (y=20)
- Main title: fontSize 32, center, strokeColor #1e1e1e
- Subtitle: fontSize 16, center, strokeColor #868e96

### 2. INTERVIEWER PROMPT (y=120)
- Rectangle: bg #f8f9fa (light gray), stroke #1e1e1e, height ~130
- Label: "INTERVIEWER PROMPT", fontSize 18
- Content: 2-3 sentence problem statement the interviewer would read

### 3. CUSTOMER OUTCOME (y=290)
- Rectangle: bg #ffe3e3 (light red), stroke #e03131, height ~140
- Label: strokeColor #e03131, fontSize 18
- Content: 1-2 sentence customer outcome statement

### 4. Side-by-side: FR + NFR (y=470)
- LEFT (x=50, width=580): FUNCTIONAL REQUIREMENTS
  - Rectangle: bg #d0ebff (light blue), stroke #1971c2
  - Bullet list of 4-6 FRs
- RIGHT (x=650, width=580): NON-FUNCTIONAL REQUIREMENTS
  - Rectangle: bg #b2f2bb (light green), stroke #2f9e44
  - Table format: Requirement | Target | Trade-off

### 5. Side-by-side: ENTITIES + APIs (y=790)
- LEFT (x=50, width=580): ENTITIES
  - Rectangle: bg #e5dbff (light purple), stroke #7048e8
  - Entity names with key fields
- RIGHT (x=650, width=580): APIs
  - Rectangle: bg #ffe8cc (light orange), stroke #e8590c
  - Key API endpoints

### 6. TECH STACK (y=1080, full width)
- Rectangle: bg #d0ebff (light blue), stroke #1971c2, height ~220
- Two columns: Data Plane | Control Plane
- Include "why chosen" rationale

### 7. HIGH-LEVEL ARCHITECTURE (y=1350)
- Component boxes with arrows showing data flow
- Use colors to distinguish layers (clients, data layer, app layer, storage)
- Rectangle boxes with labels for each component
- Arrow elements connecting them

### 8+ DEEP DIVES (y=2100+)
- Each deep dive: title (fontSize 22) + content boxes
- Use the same color coding as the component they relate to

## JSON Element Template
Every element needs these fields:
```json
{
  "id": "unique-id",
  "type": "rectangle|text|arrow",
  "x": 0, "y": 0,
  "width": 0, "height": 0,
  "strokeColor": "#1e1e1e",
  "backgroundColor": "transparent",
  "fillStyle": "solid",
  "strokeWidth": 1,
  "roughness": 1,
  "opacity": 100,
  "angle": 0,
  "seed": 1,
  "version": 1,
  "versionNonce": 1,
  "isDeleted": false,
  "boundElements": null,
  "updated": 1,
  "link": null,
  "locked": false,
  "groupIds": []
}
```

Rectangles add: `"roundness": {"type": 3}`
Text adds: `"fontSize", "fontFamily": 1, "textAlign"`
Arrows add: `"points": [[0,0],[dx,dy]], "startArrowhead": null, "endArrowhead": "arrow", "lastCommittedPoint": null, "startBinding": null, "endBinding": null`

{
